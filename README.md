# 🕵️ Honeypot Simulation

## 📌 Overview
This project implements a basic honeypot using Python socket programming to simulate an SSH server and capture attacker activity.

---

## ⚙️ Technologies Used
- Python (Socket Programming)
- Command Prompt / PowerShell
- Windows OS

---

## 🚀 Features
- Simulates fake SSH server
- Captures attacker IP address
- Logs username and password attempts
- Stores activity in log file

---

## 🧠 Concepts Covered
- Cybersecurity Defense
- Honeypots
- Intrusion Detection
- Attacker Behavior Analysis
